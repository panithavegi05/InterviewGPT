# InterviewGPT – AI-Powered Interview Assessment Platform
## Overview
InterviewGPT is an AI-driven interview assessment platform designed to simulate real-world technical, coding, and behavioral interviews. The system analyzes candidate resumes, generates adaptive interview questions, evaluates responses, supports voice-based answering, and produces automated hiring recommendations.
To ensure assessment integrity, the platform incorporates anti-cheat mechanisms such as tab-switch detection, inactivity monitoring, and copy-paste detection.

## Workflow
1. Candidate uploads resume
2. Skills are extracted automatically
3. Interview questions are generated dynamically
4. Candidate answers via text or voice
5. Responses are evaluated in real time
6. Anti-cheat events are recorded
7. Confidence score is generated
8. Hiring recommendation is produced
